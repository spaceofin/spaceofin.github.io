Github blog 배포 에러 처리하기

Trouble: 배포 에러

github pages를 이용한 github 블로그 배포 중 여러가지 에러가 발생하였습니다.

배포 에러 1. 타입 에러

generateStaticParams가 반환하는 값의 타입이 { slug: string }[]이어야 하는데, 반환되는 값에 undefined가 포함될 가능성이 있기 때문에 발생하는 오류입니다.

수정 전 코드
export async function generateStaticParams() {
  const slugs = await getPostTitles({ page: "commit-logs" });

  return slugs?.map((slug) => ({ slug }));
}
수정 후 코드
export async function generateStaticParams() {
  const slugs = await getPostTitles({ page: "commit-logs" });

  return (slugs ?? []).map((slug) => ({ slug }));
}

배포 에러 2. React Hook 함수의 부적절한 사용

배포 에러 2-1.

useMDXComponents 라는 React Hook이 getMetadata라는 함수 내에서 호출되고 있는 것이 문제입니다. useMDXComponents 함수의 함수명을 getMDXComponent로 수정합니다.

수정 전 코드
// lib/getMetadata.ts
import { compileMDX, CompileMDXResult } from "next-mdx-remote/rsc";
import { Frontmatter } from "@/types/mdx-post";
import { useMDXComponents } from "@/mdx-components";

export async function getMetadata({
  source,
}: {
  source: string;
}): Promise<CompileMDXResult<Frontmatter>> {
  return await compileMDX({
    source,
    options: {
      parseFrontmatter: true,
    },
    components: useMDXComponents({}),
  });
}
수정 후 코드
// lib/getMetadata.ts
import { compileMDX, CompileMDXResult } from "next-mdx-remote/rsc";
import { Frontmatter } from "@/types/mdx-post";
import { getMDXComponents } from "@/mdx-components";

export async function getMetadata({
  source,
}: {
  source: string;
}): Promise<CompileMDXResult<Frontmatter>> {
  return await compileMDX({
    source,
    options: {
      parseFrontmatter: true,
    },
    components: getMDXComponents({}),
  });
}

배포 에러 2-2.

error 변수가 사용되고 있지 않아서 에러입니다.
에러 변수를 사용하도록 코드를 수정합니다.

수정 전 코드
// lib/getPosts.ts
export async function getPostTitles({ page }: { page: string }) {
  const { data, error } = await supabase.storage
    .from("posts")
    .list(`${page}/md`);

  const fileList = data?.map((file) => file.name.replace(/\.(md|mdx)$/, ""));

  return fileList;
}
수정 후 코드
export async function getPostTitles({ page }: { page: string }) {
  const { data, error } = await supabase.storage
    .from("posts")
    .list(`${page}/md`);

  if (error) {
    throw new Error(`Error fetching post titles: ${error.message}`);
  }

  const fileList = data?.map((file) => file.name.replace(/\.(md|mdx)$/, ""));

  return fileList;
}
